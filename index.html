Weather Analysis by Latitude

How does Latitude affect weather?

In order to answer the question above, a number of cities across a variety of latitudes were sourced using the python "citipy" library. 500+ cities were selected, and a script was used to pull weather data at these locations from the OpenWeatherMap API, focusing on four specific criteria: maximum temperature for the day, humidity, cloudiness, and wind speed. This data was pulled on February 26, 2018, and is available under data. Matplotlib was used to graph the data across all cities by latitude, and a number of trends were observed. All plots are available under comparison, and each specific analysis is available in the drop down menu above, or by clicking any of the plots presented below. The script used to pull this data is available on github.
